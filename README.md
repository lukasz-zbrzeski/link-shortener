# Link shortener - Make short your link!

This application makes your link shorter. You can paste your link into input
field and click "Create short link!". Then you will see short link. If you click
on showed link, the application will redirect you to the page with your longer link that you paste.

## Used technologies
To develop this application I used:
* Java 11
* Spring Boot 2.3.2
* Spring Boot Starter Data JPA
* Spring Boot Starter Thymeleaf
* H2 Database 1.4.200
* Lombok 1.18.12
